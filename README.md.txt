# Fingerprint ATM Flask Project 💳

This is a simple and secure banking application built with Flask. It simulates ATM operations with fingerprint-based login (or username simulation) and includes features like deposit, withdrawal, balance view, PDF receipts, and admin management.

## 🚀 Features

- User Registration and Login
- Deposit and Withdraw Money
- View Current Balance
- Downloadable PDF Receipt after transactions
- Email Alerts (after deposit/withdraw)
- Change Password functionality
- Admin Panel to manage users and transactions
- Simple session-based security
- Clean HTML templates and top navbar


## 🛠️ Setup Instructions

1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/fingerprint-ATM-System.git
   cd fingerprint-ATM-System

2. Install dependencies
pip install -r requirements.txt

3. Run the app:
python app.py

4. Open your browser at http://127.0.0.1:5000/
